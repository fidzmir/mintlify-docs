This guide walks you through everything you need to go from zero to a fully working M-TPM Portal installation — with your Gemini API key configured, your first maintenance event logged, and your first asset tag scanned. If you follow each step in order, you should be up and running in under 10 minutes.

<Tip>
  You can get a free Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey) in under two minutes. No billing setup is required to access the free tier, which is more than sufficient for evaluating and using M-TPM Portal's AI auditing features.
</Tip>

## Prerequisites

Before you begin, make sure you have the following ready:

- **Android device or emulator** running API level 26 (Android 8.0 Oreo) or higher
- **Android Studio** (latest stable release recommended) installed on your development machine
- **Gemini API key** from [Google AI Studio](https://aistudio.google.com/app/apikey)
- The M-TPM Portal project source code (cloned or downloaded to your machine)

---

## Setup Steps

<Steps>
  <Step title="Install the App via Android Studio">
    Open Android Studio and select **File → Open**, then navigate to the root folder of the M-TPM Portal project and click **OK**. Android Studio will index the project and begin resolving Gradle dependencies — this may take a minute or two on first open.

    Once the build is ready, connect your Android device via USB (with USB Debugging enabled in Developer Options) or launch an Android Virtual Device (AVD) from the **Device Manager** panel.

    <Warning>
      Before running the app, you must remove the `signingConfig = signingConfigs.getByName("debugConfig")` line from your `app/build.gradle.kts` file. This line references a signing configuration that is not included in the public repository. Leaving it in will cause the build to fail with a configuration resolution error.
    </Warning>

    After removing that line, click the **Run** button (▶) in the Android Studio toolbar or press `Shift + F10`. Select your target device from the deployment dialog and wait for the app to install and launch.
  </Step>

  <Step title="Set Your Gemini API Key">
    When M-TPM Portal launches for the first time, navigate to the **Settings** screen (accessible from the bottom navigation bar or the side drawer menu). Locate the **Gemini API Key** field and paste in your key from Google AI Studio.

    Tap **Save** to store the key. M-TPM Portal encrypts and stores the key locally on your device — it is never transmitted to any server other than Google's Gemini API endpoint when an AI auditing request is made.

    Once saved, the AI auditing features (label scanning, OCR validation) will become active throughout the app.
  </Step>

  <Step title="Explore the Dashboard">
    After configuration, you will land on the **Dashboard** — the home screen of M-TPM Portal. Take a moment to orient yourself:

    - **Watch panel** (top section) — Displays a live summary of active assets, any open or overdue maintenance tickets, and recent tag activity.
    - **Quick Actions bar** — Provides one-tap shortcuts to log a new maintenance event, scan a tag, or open the latest report.
    - **Recent Activity feed** — Shows a timestamped list of the most recent maintenance logs and tag scans recorded in the system.
    - **Bottom navigation bar** — Switches between the three core pillars: Watch, Input, and Explore.
  </Step>

  <Step title="Log Your First Maintenance Event">
    Tap the **+** button in the Quick Actions bar or navigate to the **Input** tab from the bottom navigation bar. You will be presented with the maintenance log entry form.

    Fill in the required fields:

    - **Asset ID** — Enter or scan the identifier of the asset being serviced.
    - **Event Type** — Select the appropriate category (e.g., Inspection, Repair, Lubrication, Replacement).
    - **Description** — Add a brief description of the work performed.
    - **Technician** — Confirm or update the assigned technician name.
    - **Date / Time** — Defaults to the current timestamp; adjust if you are logging a past event.

    Tap **Submit** to save the log entry. You will see a confirmation message and the event will immediately appear in the Recent Activity feed on the Dashboard.
  </Step>

  <Step title="Scan Your First Asset Tag">
    Navigate to the **Tags** section from the bottom navigation bar or the side drawer. You will see a list of all registered asset tags in the system.

    Tap the **Scan** button (camera icon) in the top-right corner of the Tags screen. Point your device camera at a physical asset label — M-TPM Portal will use Gemini-powered OCR to read the label and extract key data fields automatically.

    Review the extracted data in the confirmation dialog. If the information is correct, tap **Confirm** to register the scan against that asset's record. If the OCR result needs correction, you can edit any field before confirming.
  </Step>
</Steps>
